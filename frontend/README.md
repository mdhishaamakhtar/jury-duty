# Jury Duty Frontend

The user interface for the Jury Duty data labeling engine.

## Overview

A responsive, high-performance web interface built with SvelteKit 5 and Tailwind CSS. This frontend allows contributors to rapidly label data points with a focus on speed and accuracy.

## Key Features

- **Rapid Labeling**: Optimized for quick decision-making with keyboard shortcuts
- **Real-time Feedback**: Instant UI updates and secure authentication
- **Responsive Design**: Works seamlessly across mobile, tablet, and desktop
- **Type-Safe**: Developed using TypeScript for robust data handling

## Developing

1. **Install dependencies**:
   ```sh
   npm install
   ```

2. **Configure environment**:
   Create a `.env.local` file with your Supabase credentials.

3. **Start development server**:
   ```sh
   npm run dev
   ```

## Building

To create a production version of your app:

```sh
npm run build
```

You can preview the production build with `npm run preview`.
